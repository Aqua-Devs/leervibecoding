Wat is een API key?

Een API key is een persoonlijke sleutel waarmee jij toegang krijgt tot de AI van OpenAI (ChatGPT). Het is als een wachtwoord dat alleen jij hebt.

Veiligheid & Privacy

Jouw key wordt NIET opgeslagen op onze servers. De key blijft alleen in je browser-sessie zolang je de website gebruikt. Zodra je de browser sluit, is de key weg. Wij hebben geen toegang tot jouw key of je OpenAI account.

Hoe krijg ik een API key?

  1. Ga naar platform.openai.com/api-keys
  2. Maak een account aan (of log in)
  3. Klik op "Create new secret key"
  4. Kopieer de key en plak hem hieronder
